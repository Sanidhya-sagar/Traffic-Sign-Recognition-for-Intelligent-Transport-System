# Major-Project
This is a 8th Semester major project which aims to build a robust Traffic Sign Recognition model  for Intelligent Transportation Systems (ITS) by integrating datasets for real-world adaptability and better accuracy, training using YOLOv8.

## Datasets Used

This project uses the following datasets for traffic sign detection and recognition:

### 1. YOLO Major Datase
Dataset Link: https://www.kaggle.com/datasets/paras2829/yolo-major/data

### 2. GTSRB - German Traffic Sign Recognition Benchmark
Dataset Link: https://www.kaggle.com/datasets/meowmeowmeowmeowmeow/gtsrb-german-traffic-sign
